How to use the FSM editor

The big white box is the FSM Canvas.  This is how you draw FSMs when answering questions.   Here's how to use it:

  • Add a state: double-click on the canvas
  • Add a transition: shift-drag on the canvas
  • Move something: drag it around
  • Delete something: click it and press the delete key (not the backspace key - on Mac, this is fn + delete)
  • Make accept state: double-click on an existing state
  • Type numeric subscript: put an underscore before the number (like "S_0")
  • Type greek letter: put a backslash before it (like "\beta")
