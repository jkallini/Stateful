How to use the FSM Editor

The big white box is the FSM Editor. This is how you draw FSMs when answering questions. Here's how to use it:

  • Add a state: double-click on the editor
  • Add a transition: Hold shift and drag from one state to the center of the next state
  • Add a self-transition: Hold shift and click on the state
  • Move something: drag it around
  • Delete something: click it and press the delete key (on Mac, this is fn + delete), NOT backspace
  • Make a start state: Hold shift and drag from nowhere to the center of the state
  • Make accept state: double-click on an existing state
  • Type numeric subscript: put an underscore before the number (like S_0)
  • Type greek letter: put a backslash before it (like \beta)

For a more detailed explanation with demonstrations, view the FSM Editor Tutorial.
